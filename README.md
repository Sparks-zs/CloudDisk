# CloudDisk
  Qt6实现的文件在线浏览，上传下载的程序。
  本项目服务器部署在树莓派4B上，经过测试可上传下载超过10G的大文件，上传速度可高达70MB/s， 下载速度可达46MB/s（目前为移动机械硬盘，如果更换固态硬盘，速度可进一步提升）。
